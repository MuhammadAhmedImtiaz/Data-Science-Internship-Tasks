## 🛠️ Project Tasks & Implementation

### 01. Iris Exploratory Data Analysis (Task 1)
* [cite_start]**Objective:** Load, summarize, and visualize structural patterns within the Iris dataset[cite: 9, 10, 12].
* [cite_start]**Methodology:** Utilized Pandas to inspect dataset dimensions and columns[cite: 16, 17]. [cite_start]Generated scatter plots, histograms, and box plots using Matplotlib and Seaborn to analyze feature distributions and outliers[cite: 18, 19, 20, 21, 22].

### 02. Credit Risk Prediction (Task 2)
* [cite_start]**Objective:** Predict the likelihood of a loan applicant defaulting on a loan[cite: 27, 29].
* [cite_start]**Methodology:** Handled missing data entries using median and mode imputation techniques[cite: 33]. [cite_start]Handled categorical variables and trained a Decision Tree Classifier to evaluate applicant risk[cite: 35].
* [cite_start]**Evaluation:** Evaluated model performance using an accuracy score and a confusion matrix[cite: 36].

### 03. Customer Churn Prediction (Task 3)
* [cite_start]**Objective:** Identify banking customers who are likely to leave the institution[cite: 43, 44].
* [cite_start]**Methodology:** Prepared data by applying One-Hot Encoding to categorical features such as geography and gender[cite: 48, 49]. [cite_start]Trained a supervised classification model to distinguish between active and churning profiles[cite: 50, 53, 54].
* [cite_start]**Insights:** Analyzed feature importance to isolate the primary variables driving customer attrition[cite: 51, 55].

### 04. Insurance Claim Regression (Task 4)
* [cite_start]**Objective:** Forecast medical insurance claim charges based on patient demographic attributes[cite: 56, 57].
* [cite_start]**Methodology:** Implemented a multivariate Linear Regression model tracking patient age, BMI values, and smoking status[cite: 61, 62].
* [cite_start]**Metrics:** Evaluated prediction errors using Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE) metrics[cite: 63, 68].

### 05. Personal Loan Acceptance (Task 5)
* [cite_start]**Objective:** Classify and identify customer segments highly likely to accept personal loan offers[cite: 69, 70, 71].
* [cite_start]**Methodology:** Explored target feature behaviors relative to marital status and employment sectors[cite: 75]. [cite_start]Trained a supervised Logistic Regression model to extract actionable business insights[cite: 76, 82].
